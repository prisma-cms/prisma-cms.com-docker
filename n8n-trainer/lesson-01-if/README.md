# Урок 1: Условная логика (If)

## Описание
Этот урок знакомит с базовой условной логикой в n8n.

## Что изучается
- Ручной триггер (Manual Trigger)
- Узел IF для проверки условий
- Разветвление потока данных

## Задание
Воркфлоу демонстрирует простую проверку числа:
- Если число больше 10 → выводится "Большое число"
- Если число меньше или равно 10 → выводится "Маленькое число"

## Как использовать
1. Откройте воркфлоу "Lesson 01: IF Logic"
2. Нажмите кнопку "Execute Workflow" для запуска
3. Измените значение в узле "Set Number" и запустите снова
4. Попробуйте создать свои условия
